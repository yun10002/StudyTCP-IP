## Day01
- 네트워크 프로그래밍에서 연결 요청을 허용하는 소켓의 생성과정
- 1단계) 소켓 생성 : socket() 함수호출
- 2단계) IP주소와 PORT번호 할당 : bind() 함수호출
- 3단계) 연결요청 가능상태로 변경 : listen() 함수호출
- 4단계) 연결요청에 대한 수락 : accept() 함수호출
